Funcionalidades do Site

Página Inicial (index.html)

Esta página apresenta uma visão geral do site e fornece links para outras seções.

Área do Cliente (areadocliente.html)

Nesta página, os clientes podem fazer login, visualizar e gerenciar suas informações pessoais.

Produtos (produtos.html)

Nesta página, os usuários podem navegar pelos produtos disponíveis, ver detalhes e adicionar ao carrinho de compras.

Serviços (servicos.html)

Esta página lista os serviços oferecidos pelo site, com descrições detalhadas de cada um.

Checkout (checkout.html)

Esta página permite aos usuários revisar os itens no carrinho de compras, inserir informações de pagamento e concluir a compra.
